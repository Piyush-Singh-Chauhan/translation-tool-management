# translation-tool-management
Translation Tool Management : where you can add a key and a text in English it auto generate the text in some other languages like Hindi, French, and Spenish.

# .env set up 
add variables like 
# MONGODB_URI=
# DBNAME=
# PORT=

# For Backend
cd Backend
npm install 
npm run dev 

# For Frontend 
cd Fronend 
npm install
npm run dev 

server start and you can check this on your localhost.
